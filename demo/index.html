Slick Scan Demo

This is a demo of the Slick Scan web component. It allows you to:

  • Upload images for OCR processing
  • Use your device's camera to capture images
  • View OCR results in real-time

How to Use

To use Slick Scan in your own project:

<script type="module" src="https://[your-username].github.io/[repo-name]/index.js"></script>
<slick-scan></slick-scan>
